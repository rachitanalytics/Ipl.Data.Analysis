This project performs exploratory data analysis (EDA) on Indian Premier League (IPL) datasets using Python and Jupyter Notebook. The objective is to extract meaningful insights from historical IPL matches and player performance data through data cleaning, statistical analysis, and visualizations.

Key Features
Data Cleaning and Preprocessing
Team Performance Analysis
Player Statistics Analysis
Toss Impact on Match Results
Venue-wise Performance Insights
Winning Trends and Patterns
Interactive Data Visualizations
Exploratory Data Analysis (EDA)
Technologies Used
Python
Jupyter Notebook
Pandas
NumPy
Matplotlib
Seaborn
Project Outcomes
Identified top-performing teams and players.
Analyzed the relationship between toss decisions and match outcomes.
Explored venue-specific advantages and winning patterns.
Created visual dashboards and charts for better understanding of IPL data.
Dataset

Historical IPL match and delivery datasets containing information about teams, players, venues, toss decisions, and match results.
